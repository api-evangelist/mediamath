# MediaMath GraphQL API

MediaMath (now Infillion) is a programmatic advertising platform. Their TerminalOne API covers marketer management, campaigns, strategies, supply sources, audience segments, deals, and performance data.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from MediaMath REST API
**Documentation:** https://apidocs.mediamath.com/
